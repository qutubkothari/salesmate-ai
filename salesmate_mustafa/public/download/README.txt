SAK WhatsApp Agent - Quick Start Guide
=======================================

STEP 1: Extract the ZIP
------------------------
Extract all files to a folder (e.g., C:\SAK-WhatsApp-Agent\)

STEP 2: Run START-AGENT.bat
----------------------------
Double-click START-AGENT.bat

FIRST TIME USERS:
- Browser will open automatically
- Choose 'Register' or 'Login'
- After authentication, return to the agent window
- Scan the QR code with WhatsApp

RETURNING USERS:
- QR code appears immediately
- Scan and start using

Requirements:
- Windows 10 or later (64-bit)
- Google Chrome or Microsoft Edge
- Active internet connection
- WhatsApp mobile app

How It Works:
-------------
1. Run START-AGENT.bat
2. If first time: Login/Register in browser
3. Agent saves your credentials automatically
4. Scan QR code with WhatsApp mobile app
5. Messages are processed by AI and replied automatically

Files Included:
- sak-whatsapp-agent-windows.exe (Main program)
- START-AGENT.bat (Launch agent)
- .env.example (Auto-generated config file)
- README.txt (This file)

Troubleshooting:
----------------
Q: Browser doesn't open?
A: Manually visit: http://13.62.57.240:8080/agent-login?callback=http://localhost:3001/auth-callback

Q: Can't find Chrome error?
A: Install Chrome from https://www.google.com/chrome/

Q: QR code won't scan?
A: Make sure you completed login/registration first
